= Using {paraview}

[[paraview_on_remotedesktop]]
=== On Remote desktop

In the {cesga} remote desktop to get {paraview} running:

* open a terminal,
* type: `module load gcc/6.4.0 openmpi/2.1.1 paraview/5.4.0`
